# Fullstack JS Roadmap

Roadmap học full-stack JavaScript (Node/NestJS + Next.js) chuẩn bị apply intern.

Xem các [Issues](../../issues) — mỗi issue là một phase học tập, kèm checklist kiến thức, nguồn học, dự án thực tế và acceptance criteria.

Xem [RULES.md](./RULES.md) để biết các quy tắc bắt buộc trong quá trình học.

## Các phase

- Phase 0 — Nền JS vững chắc
- Phase 1 — Node.js & Express
- Phase 2 — Database + NestJS
- Phase 3 — React + Next.js
- Phase 4 — Dự án tổng hợp + intern prep
